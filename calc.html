How I'd like to extend this in the future

My next goal is to make the calculator draggable. I'd then like to make it a browser extension, so users can have a draggable calculator on a webpage. Currently if I want to do this, I need to use the calculator app which disappears once I click on the actual webpage. I'd also like to have a button on the calculator to make notes, with the saved calculation and a link to the website. I think this could be a fun project to build beyond the utilit of a calculator.
